原理解析

所有文件会使用 Cloudflare 加速跳转，文件会跳转至 JSDelivr，Git 会进行重定向到 Fastgit.org

注意：公共免费资源，请合理、适度使用 (*≧ω≦)

加速模式

  • Git 仓库克隆：点击按钮复制 git clone $URL 到剪贴板
  • 文件下载模式：点击按钮在新标签页加速下载

支持的地址

  • 分支源码：archive/master.zip
  • Release源码：archive/v0.1.0.tar.gz
  • Release文件：releases/download/v0.1.0/example.zip
  • 分支文件：blob/master/filename
